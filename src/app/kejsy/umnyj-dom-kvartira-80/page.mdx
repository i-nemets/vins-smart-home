# Кейс: умный дом в квартире 80 м²

## Задача
Автоматизировать освещение, климат, безопасность (протечки/дым), управление с телефона.

## Решение
- **Zigbee**: 15 устройств (лампы, датчики движения/протечки/дым).
- **Хаб**: Home Assistant на Raspberry Pi 4, интеграция HomeKit.
- **Сценарии**: автовыключение света, климат по расписанию, алерты на телефон.

## Результат
- Снижение энергопотребления ~10% (свет + климат).
- Удобство: управление одной кнопкой или голосом (Siri).
- Безопасность: оповещения о протечках/дыме за секунды.

# Кейс: автоматизация офиса 200 м²

## Задача
Автоматизировать свет (5 зон), климат, СКУД и интеграция с бухгалтерией (время прихода/ухода).

## Решение
- **KNX**: освещение и климат по зонам, присутствие/движение.
- **СКУД**: считыватели RFID, интеграция с 1С через API.
- **Панель**: сенсорная на входе + мобильное приложение.

## Результат
- Экономия энергии ~12% (автовыключение + диммирование).
- Удобство: управление по зонам, авто‑режимы.
- СКУД: учёт рабочего времени для бухгалтерии.

# Кейс: IoT-мониторинг склада

## Задача
Склад (2000 м², фарма) требовал контроль температуры/влажности 24/7 с алертами и отчётами.

## Решение
- **Датчики**: 12× SHT31 (температура/влажность), LoRaWAN.
- **Шлюз**: Mikrotik + ChirpStack, MQTT-брокер.
- **Панель**: Grafana + алерты в Telegram.
- **Хранение**: InfluxDB, ретеншн 2 года.

## Результат
- Снижение потерь товара на 8% (раннее выявление отклонений).
- Автоматические отчёты для аудита.
- Время реакции на аномалию < 3 мин.

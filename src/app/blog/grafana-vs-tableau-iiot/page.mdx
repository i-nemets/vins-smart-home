# Grafana vs Tableau для IIoT

## Grafana
- Open-source, бесплатный (или Enterprise)
- Интеграции с TimescaleDB/InfluxDB/Prometheus
- Real-time дашборды, алерты
- Кривая обучения: средняя

## Tableau
- Коммерческий, мощная аналитика
- Drag&drop, подходит для бизнес-аналитики
- Дорого, сложнее для real-time

Для IIoT чаще выбирают Grafana (live-данные, открытость, cost-effective).

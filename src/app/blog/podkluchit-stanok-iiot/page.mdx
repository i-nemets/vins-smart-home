# Как подключить станок к IIoT

1. **Аудит**: протоколы (OPC UA/Modbus/TCP), доступные сигналы.
2. **Шлюз/конвертер**: преобразование в MQTT или прямая запись в БД.
3. **Хранение**: TimescaleDB/InfluxDB для телеметрии.
4. **Панель**: Grafana/собственный дашборд по ролям.
5. **Тестирование**: проверка стабильности и точности данных.

# Сетевая инфраструктура умного дома

## Роутинг и VLAN
Отдельные сегменты для IoT, гостевой сети, основной. Firewall-правила.

## PoE
Питание по Ethernet для камер, точек доступа, панелей.

## Точки доступа
Mesh или контроллер + AP для бесшовного роуминга.

## Резервирование
ИБП для критичных узлов (роутер, сервер, стойка).

Хорошая сеть — основа стабильности умного дома.

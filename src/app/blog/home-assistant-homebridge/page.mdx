# Home Assistant vs Homebridge

## Home Assistant
Мощный хаб с тысячами интеграций, автоматизации, панели. Требует времени на настройку.

## Homebridge
Лёгкий мост для HomeKit: позволяет добавить не‑HomeKit устройства в экосистему Apple.

Можно комбинировать: Home Assistant как центр, Homebridge для HomeKit-доступа.

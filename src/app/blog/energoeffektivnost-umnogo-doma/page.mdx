# Энергоэффективность умного дома

## Освещение
Автовыключение, диммирование, датчики движения → 10–12% экономии.

## Климат
Зональный контроль, снижение температуры при отсутствии → 8–10%.

## Мониторинг
Смарт‑розетки и счётчики показывают «пожирателей» энергии.

Итого: 12–15% снижение счетов при грамотных сценариях.

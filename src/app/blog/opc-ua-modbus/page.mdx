# OPC UA и Modbus для IIoT

- **Modbus TCP/RTU**: простой, широко распространён, подходит для базового сбора (регистры, coils).
- **OPC UA**: современный, структурированные данные, безопасность, подписки. Для сложных систем и интеграций.

Когда что выбрать
- Modbus: старое оборудование, простой сбор, legacy.
- OPC UA: новые линии, интеграции MES/SCADA, безопасность.
